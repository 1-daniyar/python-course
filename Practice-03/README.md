# Practice 3: Python Functions and Object-Oriented Programming

This repository contains working examples for Python functions, lambda expressions,
classes/objects, and inheritance.

## Structure

```
Practice-03/
├── functions/
│   ├── basic_functions.py      # defining and calling functions
│   ├── function_arguments.py   # positional, default, keyword args
│   ├── return_values.py        # return statements and values
│   └── args_kwargs.py          # *args and **kwargs
├── lambda/
│   ├── lambda_basics.py        # lambda syntax
│   ├── lambda_with_map.py      # lambda + map()
│   ├── lambda_with_filter.py   # lambda + filter()
│   └── lambda_with_sorted.py   # lambda + sorted()
├── classes/
│   ├── class_definition.py     # classes and objects
│   ├── init_method.py          # __init__ constructor and self
│   ├── class_methods.py        # instance methods, modify/delete props
│   └── class_variables.py      # class vs instance variables
├── inheritance/
│   ├── inheritance_basics.py   # parent/child classes
│   ├── super_function.py       # super()
│   ├── method_overriding.py    # method overriding
│   └── multiple_inheritance.py # multiple inheritance
└── README.md
```

## How to run

Run any file from the Practice-03 folder:

```
python3 functions/basic_functions.py
```

## Topics covered

- Function definition, arguments, return values, *args/**kwargs
- Lambda expressions with map(), filter(), and sorted()
- Classes, objects, __init__, self, instance vs class variables
- Inheritance, super(), method overriding, multiple inheritance
